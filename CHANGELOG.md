# Changelog

## 2026-06-10（快退按钮 + Word 导入工作流）

- 跟读卡片与模拟面试页各新增「快退 3 秒」按钮（圆弧箭头 + 数字 3 图标）：mp3 播放中点一下回退 3 秒；浏览器 TTS 不支持定位时给出提示。
- 新增 `IMPORT.md`：Word（.docx）导入新问题的 agent 标准操作手册（读取 → 整理卡片 → 写入题库 → node 校验 → OpenAI TTS 生成语音），含 id 规则与常见坑。
- 新增项目级 `CLAUDE.md`：任何 agent 在本目录工作时自动加载导入流程、硬规则与设计规范，用户只需丢 Word 文档说一句导入到哪。
- README 补全「设计语言」一节（试卷风色板与硬约束，此前 CHANGELOG 引用但缺失），并把第五节改为指向 `IMPORT.md` 的简化说明。

## 2026-06-10（试卷风换肤）

- 整站 UI 换为「复古学术试卷」风格（纯视觉，布局与功能不变）：暖米黄纸面 + 深藏青墨色，零圆角、零渐变、零模糊。
- 容器一律 2px 墨色边框 + 硬偏移阴影（5px 5px 0）；卡片两段式（灰米头条 + 卡体），问题前 4px 墨色竖条。
- 状态改为「实心方块 + 文字」语义指示（藏蓝/砖红/橄榄金小面积使用），不再用胶囊徽章和彩色圆点。
- 设计规范已固化，后续 UI 改动需遵守（色板与硬约束见 README 设计语言一节）。

## 2026-06-10（界面重构）

- 整体重做为移动端优先的卡片应用：底部导航（跟读 / 面试 / 思路），跟读改为全屏卡片组，左右滑动切题，下一张卡在边缘露出。
- 卡片内常驻显示回答（可滚动）；操作键纯图标化（只听答案 / 播放问→答 / 停止 / 循环）。
- 新增循环播放模式：播完当前卡自动滑到下一张接着播，到底循环回第一题。
- 滑动切卡时语音无缝接力到当前卡片；未播放时滑动保持安静。
- 题目总览底部抽屉（点顶栏「n/N」打开，绿点=已写回答），播放设置（语速/重复/间隔）收进抽屉。
- 视觉刷新：去描边浮卡 + 柔和投影、鲜亮蓝 #2f6bff 同色相渐变、顶部淡蓝光晕、分页圆点进度。
